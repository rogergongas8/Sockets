Guía de Prueba Rápida:

1. Chat: Lanza servidor y abre 2 clientes. Escribe en uno y verás que llega al otro.

2. Tickets (Bloqueo): Lanza "Bloqueante". Abre un cliente y NO envíes nada. Abre un 2º cliente y verás que no responde (bloqueado).

3. Sync (Error): Lanza "Inseguro" y dale a "Stress Test". Verás tickets repetidos (ej: dos #5).

4. Apache vs Nginx: Lanza ambos y mira cómo Apache crea un hilo nuevo por cada cliente, mientras Nginx usa el mismo hilo.

5. Apache Pro (8888): Es un servidor "real". Lee un archivo config.txt y sirve el index.html que ves arriba. Úsalo para demostrar cómo funciona una arquitectura web profesional.
